# EduTech App

A comprehensive educational technology platform for students and teachers.

## Features

### Student Features

- Dashboard with class overview
- Assignment submission (text, drawing, file upload)
- Exam taking and results
- Note-taking system
- Materials download
- Grade tracking

### Teacher Features

- Class management
- Assignment creation and grading
- Exam creation
- Materials upload
- Student progress tracking

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL with Prisma ORM
- **Frontend**: EJS templates, Bootstrap 5
- **File Upload**: Multer
- **Authentication**: Session-based
